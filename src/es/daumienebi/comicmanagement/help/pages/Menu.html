Menú

The top menu bar consists of the following items

  • Inicio: Shows the options to exit the app or refresh the data tables
  • Conexión: Opción para establecer los valores de conexiones para el servidor FTP,la API etc y otros valores necesarias. Ajustes de conexión
  • Colecciones:Opcion para acceder a la gestión de las colecciónes Gestión de colecciones
  • Comics: Apartado para gestionar los comics (Alta,Baja...etc)Gestión de comics
  • Informes: La opcion del menu para acceder a los informes incrustados en la applicación. Puedes revisar la Ayuda de informes
  • Ayuda: La configuración disponible en la applicación como cambiar el idioma por ejemplo.

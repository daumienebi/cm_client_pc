Menu Colección

Desde este menu, un usuario puede añadir,borrar,modificar y mostrar datos de una colección

  • Añadir una Colección : Interfaz para añadir una nueva colección


    Cada colección tiene una imagen(opcional) y un nombre. Pulsando en el botón inferior izquierdo se puede abrir una ventana para seleccionar una imagen

  • Gestión de Colecciones:

    Desde esta pantalla, el administrador puede editar,borrar o visualizar los detalles de una colección.



    [los detalles de cada colección se muestran realizando la acción de doble click sobre la colección de la tabla]
